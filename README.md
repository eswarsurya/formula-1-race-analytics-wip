# Formula 1 Race Analytics - Work In Progress

Active analytics project exploring Formula 1 race results, driver performance, constructor standings, qualifying outcomes, pit stops, sprint results, and latest race updates.

## Why I Built This

I built this project to turn a personal interest in Formula 1 into a practical analytics workflow using Python, FastF1, CSV data modelling, and Power BI.

## What I Implemented

The project includes historical Formula 1 datasets, a FastF1 refresh script for the latest completed race, a latest race results CSV, and a Power BI dashboard file. It is still a work in progress, and I am continuing to improve the dashboard story and analysis.

## Tools Used

Python, FastF1, Pandas, Power BI, CSV data modelling, and sports analytics.

## Project Walkthrough

A good way to explore this project is to start with the README for context, then look at the refresh script, data tables, and dashboard file as the project develops.

## Next Improvements

Planned improvements include dashboard screenshots, driver and constructor trend pages, race by race comparisons, clearer data model notes, and a notebook version for easier review.

## Project Notes

This project shows self directed learning, data refresh scripting, Power BI modelling, and the ability to build analytics around a live subject area.
