# Formula 1 Race Analytics - Work In Progress

Active analytics project exploring Formula 1 race results, driver performance, constructor standings, qualifying outcomes, pit stops, sprint results, and latest race updates.

This project is marked as work in progress because I am still improving the dashboard story and analysis. It already has usable outputs, including historical Formula 1 datasets, a FastF1 refresh script, latest race results, and a Power BI dashboard file.

## Why I Built This

Formula 1 produces structured data across drivers, constructors, circuits, races, standings, qualifying, race results, and pit stops. I built this project to turn a personal interest into a practical analytics workflow using Python, FastF1, CSV data modelling, and Power BI.

## Analytics Questions

This project looks at driver and constructor performance, the relationship between qualifying and race results, the role of pit stops and sprint outcomes, and how the latest completed race can be refreshed for dashboard use.

## What I Implemented

I created a Power BI dashboard file for Formula 1 analytics, added historical Formula 1 CSV tables, built a Python FastF1 refresh script, and saved the latest race results into a clean CSV output. The repository is intentionally labelled as work in progress because I am still adding deeper analysis and improved visuals.

## Tools Used

Python, FastF1, Pandas, Power BI, CSV data modelling, and sports analytics.

## Repository Guide

The src folder contains the latest race refresh script. The data folder contains Formula 1 CSV tables and latest race output. The dashboards folder contains the Power BI dashboard file. The requirements file lists the Python package requirement.

## Current Outputs

Current outputs include a Power BI dashboard file, a latest race refresh script, a latest race result CSV, and historical Formula 1 data tables ready for dashboard modelling.
