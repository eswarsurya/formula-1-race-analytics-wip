# Data Notes

This folder contains selected Formula 1 CSV tables used for analytics and dashboard development.

Included data areas:

- Latest race output generated by `src/update_latest_race_results.py`
- Drivers
- Constructors
- Races
- Results
- Driver standings
- Constructor standings
- Circuits
- Status
- Qualifying
- Pit stops
- Sprint results

FastF1 cache files are intentionally excluded from the public repository because they are generated locally and make the project harder to review.
