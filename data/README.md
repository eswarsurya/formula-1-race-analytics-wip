# Formula 1 Data Notes

This folder is intended for public Formula 1 CSV datasets used by the dashboard and Python refresh workflow.

Planned files include race results, drivers, constructors, circuits, standings, qualifying, pit stops, sprint results, status tables, and the latest race output generated by the refresh script.

Large or frequently changing datasets should be checked before upload. If a file is too large or not suitable for public sharing, a sample file and source note should be used instead.
