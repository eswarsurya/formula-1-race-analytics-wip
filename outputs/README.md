# Public Output Evidence

This folder contains public-facing evidence for the Formula 1 Race Analytics WIP project.

## Files

- `dataset_inventory.csv` summarises the Formula 1 tables available locally for dashboard modelling.
- `latest_race_results.csv` contains the latest race output generated by the refresh workflow.
- `latest_race_podium_and_points.csv` gives a compact latest-race points view.
- `recent_season_race_counts.csv` summarises recent season race counts.
- `top_result_statuses.csv` summarises common race result statuses.
- `fastest_average_pit_stop_driver_ids.csv` is an early pit-stop analysis output.
- `race_analytics_output_summary.md` explains the current project direction and analytics value.

## WIP Note

This is an active project. Current evidence focuses on data refresh, data modelling, and early dashboard-ready summaries.
